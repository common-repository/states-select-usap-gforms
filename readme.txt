== Description ==

This plugin adds an option to have Associated Press US state abbreviation values for the states select in Gravity Forms.

More details on Associated Press style abbreviations here:
https://en.wikipedia.org/wiki/List_of_U.S._state_abbreviations

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. When modifying your form, choose the 'United States Associated Press' Address Type.
